# Production Planning System — MRP Model (Excel + Power Query)

This project implements a Material Requirements Planning (MRP) system built in Excel with Power Query for a fictional liquid e-cigarette manufacturer. The model processes shop and retail outlet orders, applies product recipes to calculate raw material demand, checks warehouse stock levels, and calculates reorder points based on lead time and safety stock buffers. Output includes production gaps, material shortages, transport requirements, and inventory valuation in PLN.

**Tools:** Excel, Power Query  
**Data:** Simulated — orders, recipes, warehouse stock, transport priorities
